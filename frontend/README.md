# 前端开发说明

## 功能概述

### 核心功能
- ✅ 高对比度黑白登录界面
- ✅ 员工编号仅允许输入数字（自动过滤非数字字符）
- ✅ 支持模拟模式（MOCK_MODE），无需后端即可调试
- ✅ 完整的错误处理和网络异常提示
- ✅ 登录成功后展示员工详细信息

### 用户体验优化
- ✅ 页面加载时自动聚焦到员工编号输入框
- ✅ 支持回车键快速切换输入框和提交表单
- ✅ 输入时自动清除之前的错误提示
- ✅ 登录成功后显示"重新登录"按钮
- ✅ 加载状态显示（带旋转动画）
- ✅ 平滑的过渡动画效果

### 响应式设计
- ✅ 移动端适配（480px 以下）
- ✅ 可访问性支持（键盘导航、焦点指示）
- ✅ 支持减少动画偏好设置

## 启动步骤

```bash
cd frontend
npm install
npm run dev
```

默认地址：`http://localhost:5173`

## 切换到真实后端

1. 编辑 `src/api.ts`
2. 将 `const MOCK_MODE = true` 改为 `false`
3. 确保后端运行在 `http://localhost:8080`

## 设计说明

### 视觉设计
- 纯黑白极简风格
- 输入框和按钮均采用圆角胶囊设计
- 深色背景配合高对比度文字
- 渐变按钮效果

### 交互设计
- 输入框聚焦时有高亮边框和阴影效果
- 按钮悬停时有轻微上浮动画
- 消息提示有滑入动画
- 加载状态有旋转动画

### 信息展示
登录成功后展示：
- 员工编号
- 姓名
- 部门ID
- 职务ID
